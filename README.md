# راوي | Rawi ✍️🤖

[Arabic Version Below](#arabic-version)

Rawi is a premium, AI-powered writing platform designed specifically for storytellers and novelists. It combines professional planning tools with modern AI assistance to help you build immersive worlds, deep characters, and engaging chapters.

## Features
- **Project-Based Organization**: Manage your novels into structured projects.
- **Chapter Workflow**: Write, edit, and organize chapters with ease.
- **Deep Character Creation**: Build comprehensive profiles for your characters.
- **AI-Powered Assistance**: Generate plots, character traits, and chapter drafts using advanced LLMs.
- **Public Library**: Share your completed works with an active community of readers.
- **Professional Export**: Export your manuscripts in various formats.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Radix UI.
- **Backend**: Node.js, Express, Passport.js (Auth), WebSocket.
- **Database**: PostgreSQL with Drizzle ORM.
- **AI**: OpenAI SDK (compatible with any OpenAI-style API).

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/rawi-ai.git
   cd rawi-ai
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file based on `.env.example`.

4. **Prepare the database**:
   ```bash
   npm run db:push
   ```

5. **Start development server**:
   ```bash
   npm run dev
   ```
